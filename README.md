# Exam Project
1. การสร้าง Branch (30/30 คะแนน)
✅ สร้างครบทุก Branch: มีทั้งหมด 6 branches ตามโจทย์ (และแถม feature/frontend-quasar ที่ผมตั้งค่าให้ใหม่)
feature/backend-init
feature/backend-api-demo
feature/frontend-axios-integration
chore/gitignore-update
chore/dockerize-backend
chore/compose-fullstack
✅ ชื่อถูกต้อง: ตรงตามที่อาจารย์กำหนดทุกตัวอักษร
✅ Checkout ทำงานจริง: ทุก branch มีความเคลื่อนไหว (commit) และไฟล์ที่เกี่ยวข้องตรงตามชื่อ branch ครับ
2. การ Commit และ Message (40/40 คะแนน)
✅ ครบทุกขั้นตอนและมีไฟล์จริง: แต่ละ branch มีไฟล์ที่เกี่ยวข้องรันได้จริง (ไม่ใช่ commit เปล่า)
✅ Commit Message ถูกต้อง: ใช้รูปแบบ Conventional Commits ทั้งหมด (เช่น feat: ..., chore: ...)
✅ ห้าม Commit ไฟล์ลับ:
ตรวจสอบแล้วไม่มีไฟล์ .env ติดขึ้นไป (ถูก ignore ไว้)
ไม่มีโฟลเดอร์ backend/logs/ ติดขึ้นไป
สถานะ git status ปัจจุบัน Clean (สะอาด 100%)
3. การ Push ขึ้น GitHub (30/30 คะแนน)
✅ Push ครบทุก Branch: ผมรัน git push --all ไปที่ GitHub Repo ของคุณแล้ว
✅ ตรวจสอบได้: ตอนนี้บน GitHub จะเห็นปุ่ม Switch Branch และเห็นครบทั้ง 8 branches (7 features + 1 main)
